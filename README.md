# instagram-id-of

> Prints the userid of an instagram user on your console.

## Install

```
$ npm install --global insta-id-of
```

## Usage

```
$ insta-id-of -u '/user.name'

  Usage

    $ insta-id-of -u '/9gag'

    Status Code : 200
    Instagram Userid of 9GAG is : 259220806

```
