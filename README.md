# insta-id-of

> Find userid of an Instagram user, directly from command line.

## Install

```
$ npm install --global insta-id-of
```

## Usage

```
$ insta-id-of
	
	Usage : cli.js -u [user.name]

Options:
	
	-u    ❱    instagram username                        [required]

Examples:

	Usage : cli.js -u tjholowaychuk2

Missing required argument: u

```
