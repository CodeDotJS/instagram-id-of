## Install

```
$ [sudo] npm install --global insta-id-of
```

__`Required Node.js version :`__ __`^4`__

## Usage

```
$ insta-id-of
	
	Usage   : insta-id-of <username>

	Example : insta-id-of iama_rishi

	Help    : insta-id-of -h --help

```
